# NASA Kepler Object Classification Pipeline 🪐🔭

![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)

## 📖 Overview
The **NASA Kepler Object Classification Pipeline** is a machine learning project designed to classify celestial objects and potential exoplanet candidates observed by the NASA Kepler space telescope. This repository contains the complete pipeline, from raw dataset processing to model training and performance evaluation.

## 📂 Project Structure
* `ml_project.py` — The core Python script containing the machine learning pipeline (data preprocessing, model training, and evaluation).
* `raw_dataset.csv` — The initial, unprocessed dataset containing Kepler observation metrics.
* `requirements.txt` — A list of Python dependencies required to run the project.
* `ml_evaluation_results.txt` — Raw output or logs detailing the numerical results of the model evaluation.
* `ml_project_evaluation_report.txt` — A comprehensive summary/report of the machine learning model's performance and classification metrics.
* `LICENSE.txt` — The open-source license (GPL-3.0) governing this repository.
